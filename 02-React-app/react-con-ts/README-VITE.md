# Levantar

```bash
yarn dev
```

fragment

Me permite regresar un fragmento de codigo se utiliza la siguiente sintaxis

Forma 1
```html
<Fragment>
<h1>Hola Ejemplo</h1>
<p>Linea 2</p>
</Fragment>
```
Forma 2
```html
<>
<h1>Hola Ejemplo</h1>
<p>Linea 2</p>
</>
```

Props

Utilizando las herramientas de desarrollo podemos inicializar estas propiedades del objeto
